# Ruzzle Multiplayer

## Overview
Ruzzle Multiplayer is a C-based game that enables players to engage in multiplayer word search challenges. The project consists of a multithreaded server and a client application, designed to facilitate smooth and efficient gameplay over a network.

## Features
- **Multithreaded Server**: Handles multiple client connections simultaneously
- **Client Application**: User-friendly interface for players to connect to the server and participate in games
- **Real-time Gameplay**: Real-time word search and scoring to keep the game engaging and competitive
